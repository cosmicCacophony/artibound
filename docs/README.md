# Artibound Documentation

This directory contains all design, analysis, and technical documentation for the Artibound game project.

## Quick Navigation

- **[DESIGN_INDEX.md](./DESIGN_INDEX.md)** - Master index of all documentation
- **[DOCUMENTATION_METADATA.md](./DOCUMENTATION_METADATA.md)** - Documentation metadata system guide

## Folder Structure

### `core/` - Core Design Documents
Fundamental game system design and philosophy:
- `MULTICOLOR_DESIGN.md` - Core system (bouncing, colors, 2-battlefield structure)
- `CARD_DESIGN_AND_COUNTERPLAY.md` - Counterplay mechanics
- `DRAFT_PHILOSOPHY_AND_DEVELOPMENT_APPROACH.md` - Draft philosophy & development approach

### `design/` - Card Design & Game Design
Card designs, examples, and design patterns:
- `ARTIFACT_CARD_ANALYSIS.md` - Artifact Foundry card analysis
- `RW_UB_CARDS.md` - RW/UB archetype card designs
- `MULTICOLOR_CARD_EXAMPLES.md` - Multicolor card examples
- `CARD_BRAINSTORM.md` - Card brainstorming
- `BATTLEFIELD_DESIGN.md` - Battlefield design
- `ITEMS_AND_BATTLEFIELD_UPGRADES.md` - Items and upgrades

### `analysis/` - Strategic Analysis & Game States
Game state analysis, balance analysis, and strategic insights:
- `EXORCISM_BOARD_STATES.md` - Exorcism positioning analysis
- `EARLY_GAME_DECISIONS.md` - Early game strategy
- `INTERESTING_GAME_STATES.md` - Game state catalog
- `MASTERY_MOMENTS.md` - Skill expression analysis
- `BALANCE_ANALYSIS_RW_VS_UBG.md` - Balance analysis

### `testing/` - Testing & Workflow
Testing plans, strategies, and workflow documentation:
- `TESTING_PLAN.md` - Comprehensive testing approach
- `FAST_TESTING_AND_DESIGN.md` - Efficient testing approaches
- `STRATEGY_TESTING.md` - Focused strategy tests

### `architecture/` - Technical Architecture
Technical documentation and development notes:
- `ARCHITECTURE.md` - Technical architecture
- `DEVELOPMENT.md` - Development notes & roadmap
- `IMPLEMENTATION_SUMMARY.md` - Implementation summaries

**Understanding draft philosophy?**

**Understanding technical architecture?**
