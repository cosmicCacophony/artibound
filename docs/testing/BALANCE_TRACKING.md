## Rune Generation Analysis

### UBG Rune Needs

**For Key Spells:**
- Void Cascade (5UUB): Need 2 blue, 1 black
- Multicolor Wrath (6UUBBGG): Need 2 of each color
- Nature's Wisdom (2UG): Need 1 blue, 1 green
- Knowledge Seeker (5BB): Need 2 black

**Minimum Rune Requirements:**
- Blue: 2 (for Void Cascade)
- Black: 2 (for Knowledge Seeker, Void Cascade)
- Green: 2 (for Multicolor Wrath)

**Rune Artifacts Needed:** 3-5 artifacts to hit these requirements

**From Playtest 1:**
- Ended with 1 blue, 5 black, 6 green runes
