## Matchup Analysis

### UBg Control vs RW Legion

**Matchup Rating:** ‚ö†Ô∏è **Unfavorable for UBg**

**Why UBg Struggles:**
1. **Requires setup time:** Needs spell boosters in play
2. **Vulnerable to aggro:** Go-wide strategies punish slow starts
3. **AOE dependence:** Needs board clears to answer tokens
4. **Draw dependence:** Bad draws = auto-loss

**Why RW Legion Excels:**
1. **Consistent strategy:** Token generation + buffs always works
2. **Board pressure:** Forces opponent to have answers
3. **Snowball potential:** Buffs multiply token value
4. **Less draw-dependent:** Any units + buff = value

**How UBg Could Win:**
- Draw spell boosters early (turns 2-4)
- Have AOE ready for token waves
- Stabilize before buffs get out of hand
- Use Titan's Wrath or similar finisher after stabilizing
