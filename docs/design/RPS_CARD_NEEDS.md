# RPS Archetype System - Card Creation Needs

> **Created:** 2024-12-15  
> **Status:** Active  
> **Relevance:** High  
> **Category:** Design  
> **Notes:** Specific card needs for RPS archetype system implementation

## Overview

This document lists specific cards needed to enable the Rock-Paper-Scissors archetype system. Cards are organized by archetype and priority.

---

## Aggro (Mono Red + Light Splashes)

### High Priority - Core Aggro Cards

#### Mono Red Aggressive Units (1-3 mana)
**Status**: ⚠️ Needs creation  
**Purpose**: Fast, low-cost threats that pressure early

**Needed Cards:**
- [ ] **1 mana, 2/1** - "Can attack towers directly" (red)
- [ ] **2 mana, 3/2** - "Haste" (red)
- [ ] **2 mana, 2/2** - "When this attacks, deal 1 damage to tower" (red)
- [ ] **3 mana, 4/2** - "Aggressive threat" (red)
- [ ] **3 mana, 3/3** - "When this enters, deal 2 damage to tower" (red)

#### Direct Damage Spells (2-3 mana)
**Status**: ⚠️ Needs creation  
**Purpose**: Burn spells for tower damage

**Needed Cards:**
- [ ] **2 mana, R** - "Deal 4 damage to target unit or tower" (red)
- [ ] **2 mana, R** - "Deal 3 damage to target unit. If it dies, deal 2 damage to tower" (red)
- [ ] **3 mana, RR** - "Deal 5 damage to target unit or tower" (red, consumes runes)

#### Combat Tricks (1-2 mana)
**Status**: ⚠️ Needs creation  
**Purpose**: Protect units, push damage

**Needed Cards:**
- [ ] **1 mana, R** - "Target unit gains +2/+0 this turn" (red)
- [ ] **1 mana, R** - "Target unit gains +1/+1 and can attack towers this turn" (red)
- [ ] **2 mana, R** - "Target unit gains +2/+2 this turn" (red)

#### Tower-Focused Units
**Status**: ⚠️ Needs creation  
**Purpose**: Units that can attack towers directly

**Needed Cards:**
- [ ] **3 mana, 2/2** - "Can attack towers directly" (red)
- [ ] **4 mana, 3/3** - "Can attack towers directly. When this attacks a tower, deal 1 damage to all enemy units" (red)

### Medium Priority - Splash Support

#### White Splash (RW)
**Status**: ⚠️ Needs creation  
**Purpose**: Combat tricks, protection, team buffs

**Needed Cards:**
- [ ] **1 mana, W** - "Target unit gains protection from spells this turn" (white)
- [ ] **2 mana, W** - "All your units gain +1/+1 this turn" (white)
- [ ] **3 mana, RW** - "All your units gain +2/+0 this turn" (red/white)

#### Black Splash (RB)
**Status**: ⚠️ Needs creation  
**Purpose**: Removal, direct damage

**Needed Cards:**
- [ ] **2 mana, B** - "Destroy target unit with 2 or less health" (black)
- [ ] **3 mana, RB** - "Deal 4 damage to target unit. If it dies, deal 2 damage to tower" (red/black)

## Implementation Priority

### Phase 1: Enable Aggro (Highest Priority)
1. Create mono red aggressive units (1-3 mana)
2. Create direct damage spells (2-3 mana)
3. Create combat tricks (1-2 mana)
4. **Optional**: 1-2 red rune artifacts (for Time of Triumph builds)
5. Test Aggro vs Control matchup

### Phase 2: Enable Midrange
1. Create black core removal (2-3 mana)
2. Create black card draw (3-4 mana)
3. Create value units (3-5 mana)
4. **Create rune artifacts** (dual-color, flexible, any-color) - **Critical for 3-4 color decks**
5. Test Midrange vs Aggro matchup

### Phase 3: Complete Control
1. Create GUW finisher
2. Create GUR finisher
3. **Create rune artifacts** (dual-color GU, flexible UB) - **Enable splashing**
4. Test Control vs Midrange matchup

### Phase 4: Enhance All Archetypes
1. Create splash support cards
2. Create multicolor value cards
3. Create 3-4 color payoffs
4. Balance test all matchups

**~53-79 cards needed** to fully enable RPS system
